## 安装

```bash
npm install -g denaro-gitckt
```

## 使用

### 命令行交互形式

```bash
gitckt
```

### 命令行参数形式

```bash
gitckt clone <url> [savePath] -b <branch> -d <dir> 
```

## 帮助命令

### 命令行交互形式

```bash
gitckt -h
```

### 命令行参数形式

```bash
gitckt clone -h
```
